## Run Locally

Clone the project

```bash
  git clone https://github.com/Beast-Rey/Yelp-Review
```

Go to the project directory

```bash
  cd Yelp-Review-main
```
To start server

Go to the server directory

```bash
  cd server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

To start client

Go to the server directory

```bash
  cd client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run preview
```
